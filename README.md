# Mini ATS

A simple Applicant Tracking System built with React, TypeScript and Supabase.

## Features

- Admin and customer accounts
- Authentication with Supabase
- Create and manage jobs
- Add candidates with LinkedIn profiles
- Kanban pipeline
- Filter candidates by job and candidate name
- AI-based candidate assessment

## Tech Stack

- React
- TypeScript
- Vite
- Supabase
